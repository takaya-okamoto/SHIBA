# 行動規範

このプロジェクトは [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
v2.1 を行動規範として採用します。

参加者は、ハラスメントのない、敬意ある協力的な環境づくりに協力してください。
容認されない行動を見聞きした場合は、GitHub 経由でメンテナに報告してください。報告は秘密として扱います。

行動規範の全文は上記リンクを参照してください。日本語訳は
[こちら](https://www.contributor-covenant.org/ja/version/2/1/code_of_conduct/) にあります。
